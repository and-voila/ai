## Open Source Stack

And Voila AI embraces open source tools for maximum transparency and collaboration. We believe open ecosystems foster innovation. Below are the key open source technologies powering our product and operations:

- **[Clerk](https://clerk.dev/)**: User authentication and management done right. Clerk provides a fully-featured login system out-of-the-box with support for email/password, social logins, and more.

- **[ContentLayer](https://www.contentlayer.dev/)**: ContentLayer enables a seamless content authoring workflow outside of React, making content management smooth.

- **[Framer Motion](https://www.framer.com/motion/)**: This animation library brings UI to life with declarative transitions and animations. Easy to use and integrate.

- **[Inngest](https://www.inngest.com/)**: A flexible orchestration layer for AI and function chaining. Inngest streamlines workflow automation.

- **[LangChain](https://langchain.readthedocs.io/)**: An open source library for chaining natural language models into custom workflows. Critical for creating advanced experiences.

- **[Neon Serverless Postgres](https://neon.tech/)**: Neon makes Postgres serverless, allowing for auto-scaling, multi-tenant isolation, and other benefits.

- **[Next JS](https://nextjs.org)**: The React framework powering our frontends with its optimized builds, routing, and DX.

- **[Nextra](https://nextra.site/)**: Our documentation built with Nextra, React + MDX based for streamlined technical writing.

- **[Novel.sh](https://novel.sh/)**: An open source AI-powered editor that helps generate content. Integrates nicely into our workflow.

- **[Prisma](https://www.prisma.io/)**: Prisma adds type safety to database access in TypeScript. Critical for maintainable data layers.

- **[Radix UI](https://radix-ui.com)**: Composable and accessible React primitives for high-quality UI without compromise.

- **[shadcn/ui](https://github.com/shadcn/ui)**: Thoughtfully designed React components and hooks for consistent, beautiful UIs.

- **[Stripe](https://stripe.com/)**: Feature-rich payments and billing management. Stripe helps us focus on core product.

- **[Tailwind CSS](https://tailwindcss.com)**: Utility-first CSS framework providing design flexibility without constraint.

- **[Tailwind UI](https://tailwindui.com/)**: Builds on Tailwind CSS with production-ready UI components. Accelerates development. This website is built with Tailwind's Studio template.

- **[Tier.run](https://tier.run/)**: Flexible usage-based billing management for pricing experiments and optimization.

- **[Turborepo](https://turborepo.com)**: Monorepo management allowing us to build apps in parallel while sharing code.

- **[TypeScript](https://www.typescriptlang.org)**: Adds optional types to JavaScript for more robust code at scale. Critical for us.

- **[Upstash](https://upstash.com)**: Managed high-performance Redis with serverless benefits like no ops management.

- **[Vercel](https://vercel.com)**: Deploys and hosts our frontends with optimized performance and developer experience.

- **[Vercel AI SDK](https://sdk.vercel.ai/docs)**: Toolkit for integrating AI seamlessly into Vercel serverless functions.
